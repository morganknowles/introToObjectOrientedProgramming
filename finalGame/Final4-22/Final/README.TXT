PROJECT TITLE:Safari Catch
PURPOSE OF PROJECT:Final Project
VERSION or DATE:April/May 2016
HOW TO START THIS PROJECT: Press Run
AUTHORS:Morgan Knowles

USER INSTRUCTIONS:
This is a game based on the scenario WBC.

Object of the game:
Try to beat your previous score in the allotted time.

Controls:
Use the left and right keyboard arrows to control the Vehicle. 

How the game ends:
Catch as many Animals as you can before the time reaches 1500.
If the Vehicle is hit by a Bomb, then the game is over and you lose.

Other:
When the Vehicle hits the scoreboard, it will go back to the bottom center starting point.
